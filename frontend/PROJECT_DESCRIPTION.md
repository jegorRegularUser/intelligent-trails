# Intelligent Trails - Умная система построения маршрутов

## Описание проекта

Intelligent Trails - это современное веб-приложение для построения интеллектуальных туристических маршрутов с интеграцией карт Яндекс. Приложение позволяет создавать как простые маршруты от точки А до точки Б, так и сложные умные маршруты с учетом интересных мест и предпочтений пользователя.

## Основные возможности

### 🗺️ Интерактивная карта
- Интеграция с Яндекс.Картами
- Отображение маршрутов в реальном времени
- Маркеры достопримечательностей и интересных мест
- Панорамирование и масштабирование

### 🚀 Два типа маршрутов

#### Простой маршрут
- Быстрое построение кратчайшего пути
- Выбор способа передвижения (пешком, велосипед, автомобиль)
- Автоматический поиск основных достопримечательностей по пути

#### Умный маршрут
- Персонализированные настройки
- Выбор типов мест для посещения:
  - 🏛️ Исторические места
  - 🎭 Культурные центры
  - 🌳 Парки и зеленые зоны
  - 🍽️ Кафе и рестораны
  - 🛍️ Торговые центры
  - 🌅 Смотровые площадки
- Настройка времени в пути
- Алгоритм оптимизации с учетом рейтингов мест

### 📱 Адаптивный интерфейс
- Современный дизайн с использованием shadcn/ui
- Адаптивная панель управления
- Прокрутка для длинных списков
- Интуитивно понятное управление

## Технологический стек

### Frontend
- **React 18** - основная библиотека UI
- **TypeScript** - типизированный JavaScript
- **Vite** - быстрая сборка и разработка
- **Tailwind CSS** - утилитарные CSS-классы
- **shadcn/ui** - компоненты интерфейса
- **Lucide React** - иконки

### Карты и геолокация
- **Яндекс.Карты API** - картографические данные
- **Геокодирование** - преобразование адресов в координаты
- **Построение маршрутов** - расчет оптимальных путей

### Архитектура
- **Custom hooks** - переиспользуемая логика
- **Component-based** - модульная архитектура
- **TypeScript interfaces** - строгая типизация
- **React Router** - навигация между страницами

## Структура проекта

```
src/
├── components/          # Переиспользуемые компоненты
│   ├── ui/             # Базовые UI компоненты
│   ├── Header.tsx      # Шапка приложения

```

## Ключевые алгоритмы

### Умная маршрутизация
- **Система оценки мест** - учет рейтинга, популярности, типа
- **Оптимизация по расстоянию** - минимизация отклонений от основного пути
- **Фильтрация по предпочтениям** - выбор мест согласно интересам пользователя
- **Ограничение по времени** - контроль продолжительности маршрута

### Геопространственные вычисления
- Расчет расстояний по формуле гаверсинуса
- Определение близости к маршруту
- Генерация промежуточных точек для визуализации

## Установка и запуск

```bash
# Клонирование репозитория
git clone <repository-url>
cd intelligent-trails

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

## Конфигурация
