# Tutorial GIFs

Эта папка содержит GIF-анимации для туториала на лендинге.

## Необходимые файлы

Добавьте следующие GIF-файлы:

1. `step1.gif` - Указание начальной точки (ввод адреса, выбор на карте, геолокация)
2. `step2.gif` - Добавление интересных мест (выбор типа места: кафе, парк, музей)
3. `step3.gif` - Выбор способа передвижения для каждого участка
4. `step4.gif` - Построение маршрута (нажатие кнопки, процесс построения)
5. `step5.gif` - Выбор из нескольких вариантов (листание карусели, смена места)
6. `step6.gif` - Кнопка "Поделиться" и сохранение маршрута

## Рекомендации

- Размер: 1280x720px или 1920x1080px
- Формат: GIF с оптимизацией
- Длительность: 5-10 секунд на анимацию
- FPS: 15-30 для плавности
- Размер файла: до 5MB на GIF

## Как записать

1. Используйте инструменты записи экрана (OBS, ScreenToGif, LICEcap)
2. Запишите действия на странице `/map`
3. Конвертируйте в GIF с оптимизацией
4. Поместите файлы в эту папку
