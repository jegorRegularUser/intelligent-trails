# 📄 Intelligent Trails - Отчёт о Производственной Практике

## 📋 Содержание

Полный отчёт о производственной практике по специальности **09.02.07 Информационные системы и программирование** с описанием разработки информационной системы **Intelligent Trails**.

- Диаграмма последовательности (Sequence Diagram)
- Диаграмма состояний (State Diagram)
- Диаграмма размещений (Deployment Diagram)

#### **Раздел 4: Разработка информационной системы** (11 подразделов)

##### 4.1 Разработка пользовательского интерфейса
- HTML5, CSS3, JavaScript ES6+
- Адаптивная верстка (Flexbox, CSS Grid)
- Интерактивная карта Yandex Maps
- Media queries для мобильных и десктопов

##### 4.2 Модуль подключения к БД
- Безопасное взаимодействие с MySQL
- Защита от SQL-инъекций (prepared statements)
- Структура БД (users, routes, route_history)

##### 4.3 Модуль аутентификации и авторизации
- Стандартная форма входа
- OAuth 2.0 с Yandex ID
- Хеширование паролей (bcrypt)
- Управление сессиями

##### 4.4 Модуль построения простых маршрутов
- Геокодирование адресов (Yandex Geocoder API)
- Построение маршрутов (Yandex Router API)
- Три режима передвижения: пешком, авто, транспорт
- Визуализация на карте в реальном времени

##### 4.5 **Интеграция с Yandex API** ⭐
- **Yandex Maps API 2.1** - отображение интерактивной карты
- **Yandex Geocoder API** - преобразование адресов в координаты
- **Yandex Router API** - построение маршрутов
- **Yandex Suggest API** - поиск мест по категориям
- Примеры JavaScript кода с инициализацией карты и маркеров

##### 4.6 **Python модуль поиска мест (yandex_api.py)** ⭐
- Класс `YandexStaticRouter` с асинхронными методами
- Асинхронная обработка (asyncio, aiohttp)
- **Multi-strategy геокодирование** (4 стратегии для 95%+ успешности)
- **Дедубликация координат** (удаление повторов)
- **Формула Хаверсина** для расчёта расстояний
- Обработка таймаутов и исключений
- Подробное логирование

##### 4.7 **FastAPI серверное приложение** ⭐
- Инициализация приложения на FastAPI
- CORS middleware для кросс-доменных запросов
- Endpoint `/api/search/places` с валидацией данных
- Асинхронная обработка (async/await)
- Производительность: 1000+ одновременных соединений

##### 4.8 **Интеграция PHP + Python** ⭐
- REST API архитектура
- Fetch API запросы из JavaScript
- Обработка асинхронных операций (Promise/async-await)
- Кэширование результатов на 10 минут
- Обработка ошибок сети и таймаутов
- Двусторонее логирование (PHP + Python)

##### 4.9 **Алгоритмы маршрутизации** ⭐
- Проблема коммивояжёра (TSP)
- Алгоритм "Ближайшего соседа" O(N²)
- 2-opt оптимизация маршрутов
- Оптимизация по временному лимиту
- Балансировка по категориям мест
- Таблица производительности алгоритмов

##### 4.10 **Интересные технические решения** ⭐
1. Асинхронное управление сессией aiohttp
2. Multi-strategy геокодирование (4 подхода)
3. Дедубликация по координатам
4. Валидация географических данных
5. Таймауты и обработка исключений
6. Логирование на всех уровнях
7. Формула Хаверсина для расчётов
8. CORS для безопасности
9. JSON в MySQL для гибкости

##### 4.11 Модуль личного кабинета пользователя
- Просмотр информации профиля
- Управление паролем
- История маршрутов (таблица)
- Статистика (всего маршрутов, расстояние, время, режим)
- Удаление маршрутов
- SQL запросы и JavaScript обработка
